jupyter主题安装

真是No Zuo No Die，原本只是觉得jupyter上的主题太亮了，想换一下主题的，结果浪费了一下午。。。大概因为我是选择困难户吧。

安装主题的过程比较简单：sudo pip install jupyterthemes

安装完后看一下已安装了哪些主题：sudo jt -l

清除复原配置： sudo jt -r

切换主题： sudo jt -t grade3

加配置的主题切换：sudo jt -t grade3 -cellw 1050 -lineh 155 -fs 11 -nfs 12 -tfs 12 -f source -tf sourcesans -nf sourcesans

其中：

其中的参数详细请看：jupyterthemes Github

不过从小编的使用情况来看，体验不好的有两点：一点是转html时字体没有跟着转过去，导致字体不是我们配置的字体，这一点后来解决了，因为可以通过修改浏览器的字体解决；另一点是主题太少啦，都没有自己很喜欢的。
