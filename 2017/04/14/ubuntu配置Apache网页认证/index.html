ubuntu配置Apache网页认证

1、目的：浏览器网页登入认证

Ubuntu上安装Apache2做web服务器比较顺畅，基本不用配置什么，大部分采用默认的配置，网页根目录默认放在/var/www/html/中。主机因配有静态IP，其他电脑直接访问IP地址就可以进入了。不过有时我们的服务器只想内部人员访问，即浏览器输入网址时，要用户帐号与密码。一开始认为用PHP等脚本在后台做认证，这种方式应该也行。不过转念一想，能在Apache上做配置，会更加方便。

2、配置过程
