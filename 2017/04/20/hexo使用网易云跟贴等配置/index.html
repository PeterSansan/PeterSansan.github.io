hexo使用网易云跟贴等配置

这几天时不时都会捣鼓一下Hexo博客的事，我个人比较喜欢默认的landscape主题，其他的主题没有看到十分合胃口的，所以我一直沿用着landscape。但它还是有一些功能很不完善的，像没有关于页面，RSS与搜索就是摆设，不支持Latex公式编译等等。

公式编辑器可以安装MathJax解决，网上教程也不少。今天我就讲一下怎样在landscape上实现跟帖功能与添加About页面。

1、网易云跟贴配置

之的以使用网易云跟贴理由是多说评论要关闭了，没有过多的选择，网上打了一下，找到了网易云跟贴，因此就试验了一下。大家可以先按[m官网]的步骤操作，

  • 1、进入站长管理平台，获取代码—WEB页面代码—点击右侧复制代码按钮，获取云跟贴接入代码

  • 2、打开项目目录，进入路径：themes/landscape/layout/_partial，找到article.ejs文件。参照下图，找到节点<footer class="article-footer">在结束标签</footer>（准确的说位于第31行）后粘贴云跟贴代码，保存，重启服务。不重启的话，就直接生成发布吧。

    问题出现 ：每个文章的页面最下面没有问题，都有评论栏，如下图

问题出在首页，它莫名其妙地在第一篇文章的后面出现了那多个上面图一样的评论栏，如果你没有出现这样的情况，那真是恭喜你了，但我是就遇到了这样的情况。怎么办呢？方法是在复贴的那段代码外面加一个条件判断。老实说，我并不懂这个ejs文件的语法，我只是参考它文件的写法，自己加了一个条件判断是不是主页，如果不是主页，就增加，是主页，就不添加，详细代码变为：

<div id="cloud-tie-wrapper" class="cloud-tie-wrapper"></div>

<script>
var cloudTieConfig = {
  url: document.location.href,
  sourceId: "",

这样问题就可以解决了。
